# 🌾 COOP-AFBE - Cooperativa Dos Produtores e Agricultores Familiares de Beberibe LTDA.

![GitHub repo size](https://img.shields.io/github/repo-size/josefilhoest/COOP-AFBE?style=for-the-badge)
![GitHub last commit](https://img.shields.io/github/last-commit/josefilhoest/COOP-AFBE?style=for-the-badge)

Este é um site institucional desenvolvido para a **COOP-AFBE**, com o objetivo de dar visibilidade aos pequenos produtores rurais de Beberibe, facilitando o contato com consumidores e promovendo a agricultura sustentável.

## 🔗 Demonstração
Você pode visualizar o projeto online aqui:  
👉 [https://josefilhoest.github.io/COOP-AFBE/](https://josefilhoest.github.io/COOP-AFBE/)

---

## 🚀 Funcionalidades
- **Navegação Inteligente:** Menu com âncoras para rolagem suave entre seções.
- **Botão de Conversão:** Integração direta com WhatsApp através de um botão flutuante com animação de pulsação.
- **SEO Otimizado:** Meta tags configuradas para melhor indexação no Google.
- **Identidade Visual:** Favicon personalizado e integração com redes sociais.
- **Design Responsivo:** Adaptado para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, animações (keyframes) e layout responsivo.
- **FontAwesome:** Biblioteca de ícones profissionais.
- **GitHub Pages:** Hospedagem gratuita e rápida.

---

## 🎨 Layout
O projeto conta com seções de:
- `Início` (Hero section)
- `Produtos` (Hortifruti e Orgânicos)
- `Sobre Nós` (História da Cooperativa)
- `Contato` (Localização e links sociais)
